# dotscale

Created by Yofre (yfrvids)

dotscale es un script que permite escalar pantallas/monitores en dwm o windows managers que usen Xorg, X11.

## Script

- Copiar y pegar el archivo dotscale en /usr/local/bin/ (`sudo cp -r dotscale /usr/local/bin/`)
- Darle permisos al archivo de script (`sudo chmod +x /usr/local/bin/dotscale`)
- Ejecutar mediante el comando `dotscale`
